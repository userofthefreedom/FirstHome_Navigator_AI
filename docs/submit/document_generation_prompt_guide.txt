제출 문서 생성을 GPT에 요청하기 위한 프롬프트 작성 가이드
작성일: 2026-06-26

목적
이 문서는 FirstHome Navigator AI 프로젝트의 제출 산출물 중 다음 문서를 GPT에게 요청할 때 어떤 식으로 프롬프트를 작성하면 좋은지 설명한다.

대상 문서
1. API 흐름도 또는 시퀀스 다이어그램
2. Vue 컴포넌트 구조도
3. DB 모델링 ERD
4. 페이지 설계 또는 화면 설계도

공통 주의사항
- GPT에게 "예쁘게 만들어줘"라고만 요청하지 말고, 반드시 입력 자료, 출력 형식, 포함할 범위, 제외할 범위를 함께 준다.
- 이 프로젝트는 Django REST Framework + Vue 3 SPA 구조라는 점을 먼저 알려준다.
- 실제 코드 기준으로 작성하라고 요청한다.
- 추측이 필요한 부분은 "추정"이라고 표시하게 한다.
- 한국어 산출물로 작성하게 한다.
- Mermaid 문법이 필요한 경우 "Mermaid로 렌더링 가능한 코드 블록"을 요구한다.
- 발표자료에 넣을 목적이면 너무 세부적인 필드 전체보다 핵심 관계를 우선하게 한다.


1. API 흐름도 또는 시퀀스 다이어그램 요청 프롬프트

추천 입력 자료
- docs/submit/docs_submit/service_architecture.txt
- frontend/src/api/firsthome.js
- backend/config/urls.py
- 각 앱 urls.py
- 대표 화면: 조건 입력, 추천 청약, 공고 상세, 자금 로드맵, AI 코치, MY PAGE

요청 예시

아래 자료를 바탕으로 FirstHome Navigator AI의 API 흐름도와 주요 시퀀스 다이어그램을 작성해줘.

프로젝트 구조:
- Frontend: Vue 3 SPA, Vue Router, Pinia, Axios
- Backend: Django REST Framework, Session Auth, SQLite
- 주요 API 클라이언트: frontend/src/api/firsthome.js
- 주요 백엔드 라우팅: backend/config/urls.py와 각 app urls.py

반드시 포함할 흐름:
1. 회원가입/로그인/세션 확인
2. 조건 입력 저장
3. 청약 추천 조회
4. 공고 상세와 주택형 옵션 조회
5. 자금 로드맵 조회
6. AI 코치 요약 생성
7. 관심 공고/옵션 저장 및 해제
8. 금융상품 조회와 가입 후보 저장
9. 청약 지도 조회
10. 청약 아고라 게시글/댓글 작성

출력 형식:
- 먼저 전체 API 흐름을 설명하는 요약 문단
- 그 다음 Mermaid sequenceDiagram 3개
  1. 조건 입력부터 추천 청약까지
  2. 공고 상세부터 자금 로드맵과 AI 코치까지
  3. MY PAGE 저장 내역 조회와 삭제 흐름
- 마지막에 API 그룹별 표를 작성

주의사항:
- 실제 코드에 없는 API는 만들지 말 것
- 엔드포인트 이름은 /api/... 형태로 정확히 적을 것
- 비회원 client id와 회원 session의 차이를 표시할 것
- AI 요청은 GMS OpenAI 호환 API를 사용하는 외부 호출로 표시할 것

5. 산출물 품질 체크리스트

GPT가 생성한 결과물은 제출 전에 다음을 확인한다.

API 흐름도
- 실제 엔드포인트와 일치하는가
- 프론트엔드 호출 함수와 백엔드 URL이 연결되는가
- 세션 인증과 비회원 client id 차이를 표시했는가
- AI 외부 API 호출이 분리되어 보이는가

Vue 컴포넌트 구조도
- App, RootLayout, Page, Component, Store, API 계층이 구분되는가
- 실제 router에 있는 화면만 포함했는가
- 공통 챗봇과 테마 전환이 포함되었는가

DB ERD
- 실제 models.py 기반인가
- User, Profile, Notice, HousingUnitOption, FinancialProduct, Favorite, Community 관계가 빠지지 않았는가
- 발표용 핵심 관계가 한눈에 보이는가

페이지 설계도
- 사용자 여정이 자연스러운가
- 비회원/회원 차이가 설명되는가
- 화면별 목적과 다음 행동이 명확한가
- 캡처 화면과 설명이 어긋나지 않는가
