# FirstHome Beta Test Plan

## 목표

실제 발표/시연과 동일하게 공개 URL에 접속해 사용자가 보는 화면, 조작, 응답, 성능, AI 챗봇 맥락을 검증한다.

## 범위

1. 비로그인 전체 플로우
   - 대시보드
   - 조건 입력
   - 추천 청약
   - 공고 상세
   - 청약 지도
   - 자금 로드맵
   - AI 코치 로그인 유도
   - 금융상품
   - 경제 NOW
   - 청약 아고라
   - MY PAGE 로그인 유도
   - 전역 AI 챗봇

2. 회원가입/로그인 전체 플로우
   - 실제 UI 회원가입
   - 회원가입 직후 조건 입력 화면 이동
   - 이름/출생년도 기본값 반영 확인
   - 조건 저장
   - 추천 후보 선택
   - 관심 공고/옵션 저장 및 해제
   - 금융상품 가입 후보 저장 및 해제
   - MY PAGE 저장 목록/그래프 확인
   - AI 코치 심층 화면 확인
   - 전역 AI 챗봇 화면별 응답 확인
   - 로그아웃 후 비로그인 상태 복귀

3. 30인 동시 접속
   - 30개 독립 브라우저 컨텍스트
   - 각 세션은 홈, 추천, 상세, 자금, 금융상품, 챗봇 API를 순차 이용
   - 응답 시간, 실패율, 4xx/5xx, 콘솔 오류 수집

## 수집 항목

- 페이지별 `domcontentloaded`, `networkidle` 소요 시간
- 콘솔 오류와 페이지 예외
- 실패한 네트워크 요청
- 주요 버튼 클릭 결과
- 챗봇 질문/응답 본문 일부
- 주요 화면 스크린샷
- 30인 동시 접속 성공률과 p50/p95 응답 시간

## ngrok 처리

유료 ngrok 도메인을 사용하므로 테스트 브라우저와 API 요청에 별도 우회 헤더를 넣지 않는다. 베타 테스트는 발표 관객이 실제로 접속하는 조건과 최대한 동일하게 공개 URL을 직접 검증한다.
