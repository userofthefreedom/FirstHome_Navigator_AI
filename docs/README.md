# Documentation

FirstHome Navigator AI 문서는 용도별로 아래 폴더에 정리했습니다.

## Folder Guide

| Folder | Contents |
|---|---|
| `history/` | 과거 MVP/확장 기획서 버전 문서 |
| `guide/` | 실행 가이드, API Key 신청 가이드, 외부 서비스 신청 안내 |
| `detail/` | ERD, Use Case, 와이어프레임, WBS, 규칙 상세 설명, 참고 프로젝트 문서 등 상세 자료 |

최신 실행 방법은 프로젝트 루트의 `README.md`를 우선 확인합니다.

`detail/rules_detail.txt`는 추천 점수, 자금 계산, 지역 매칭, 공고 분류, PDF 분석, 금융/대출/정책 후보 선정, AI 안전 규칙을 설명하는 상세 자료입니다.
